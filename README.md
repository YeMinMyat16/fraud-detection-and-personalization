# Financial Fraud Detection & Personalization System

## Overview
This project focuses on detecting fraudulent financial transactions
and generating personalized insights using machine learning and
data mining techniques on large-scale transaction datasets.

## Dataset
- Credit Card Fraud Dataset
- Bank Transaction Dataset
(~480,000 transactions)

## Techniques Used
- DBSCAN clustering
- Market Basket Analysis
- Hybrid recommender systems
- Anomaly detection models

## Evaluation Metrics
- ROC-AUC
- F1-score
- Silhouette score
- Support, confidence, lift

## Key Insights
- Identification of high-risk transaction patterns
- Detection of anomalous behavior across devices and time windows

## What I Learned
- Handling large transactional datasets
- Evaluating unsupervised and hybrid models
- Interpreting ML metrics for real-world problems
